計分系統

網球比賽的計分單位有四個，分別是：

  • 分（point）
  • 局（game）
  • 盤（set）
  • 場（match）

    • 單、雙打計分方式一樣。（以下皆以三盤二勝制為例做說明）

  • 四分（point） 贏得一局（game）
  • 六局（game） 贏得一盤（set）
  • 二盤（set） 贏得比賽（match）

    • 比分範例

    Game Set 1 2 3
    球員 A 40 1 6 2 3
    球員 B 15 1 4 6 0
    第一盤 Set 1

    球員 A 贏得此盤

    局數 6-4

    第二盤 Set 2

    球員 B 贏得此盤

    局數 6-2

    第三盤 Set 3

    球員 A 在第四局發球

    局數 3-0

    分 point

    一場網球比賽 (match) 一般由一至五盤 (set) 比賽構成，而每一盤 (set) 一般又分成數局 (game)，運動員通過得分 (point) 來贏得局 (game)，率先贏得規定局數的運動員贏得一盤，而率先贏得規定盤數的運動員則贏得比賽。

    得分狀態 分數呼叫
    0 points 無得分 “Love”
    1 point 第一分 “15” fifteen
    2 points 第二分 “30” thirty
    3 points 第三分 “40” forty
    贏得該局 “Game”
    Tied score 雙方分數平手 "All"
    40-40 平局 “Deuce”

    局 game

    一局（game）比賽只有一名運動員發球，率先贏得至少 4 分並多出對手至少 2 分的運動員贏得一局。一局比賽中，每一次發球必須在半場的兩個發球區輪流，每局第一次發球應先從右邊開始。

    網球每局的記分方法：從 0 至 3 分分別為「零」（love）、「十五」（fifteen）、「三十」（thirty）和「四十」（forty）。記分時，發球手的得分在前。因此「30 比 0」的意思是，發球手贏得 2 分，而接球手還未得分。

    平局 Deuce

    當雙方運動員都得到了 3 分（40 比 40）時，一般叫「平局」（deuce）而非 (forty-all)。

    在出現平局後一名球手再得一分，被稱為「占先」（advantage），而不再記分。如果在占先的情況下失去一分，就再度回到平局；如占先後再得一分，就贏得一局。

    一局比分範例 分數呼叫
    40-0 forty-love
    30-40 thirty-forty
    15-15 fifteen-all
    40-40 deuce
    一局比分試算
    Game Set 1 2 3
    球員 A 0 0 0 0 0
    球員 B 0 0 0 0 0

    盤 set

    盤的計算有多種，基本上，率先獲得 6 局 並領先對手至少 2 局的運動員贏得該盤；若其中一方贏得 6 局卻未領先至少 2 局，則該盤必須繼續，直到一方領先 2 局為止。

    最常見的是「領先盤」(advantage set）和「決勝盤」(tie-break set) 兩種。

    領先盤 advantage set

    即 long game，首先贏得 6 局或 5 平後領先差距 2 局為止。 這種計分方式在職業賽中時，常僅出現於最後一盤（五盤三勝比賽中之第五盤，或三盤兩勝比賽中之第三盤）。

    決勝局／盤 tie-break set

    另一常見的記分方法是，當雙方在一盤中戰至 6 比 6 時，會舉行一局特殊的「決勝局」，又稱「搶七」。贏得此決勝局的運動員就以總局數 7 比 6 贏得該盤。

    在搶七中，首先獲得 7 分且淨勝 2 分以上的運動員贏得該盤比賽。由於在決勝局中雙方交替發球，無論哪一方獲勝該局均不會被記入保發/破發局。 決勝局以 0,1,2,3... 呼叫，且比分後面呼叫領先者姓氏，首先贏得 7 分並有 2 分差距即贏得該局及盤。

    搶七比分範例 分數呼叫
    1-0 one-zero
    4-3 four-three
    4-4 four-all
    3-7 set

    決勝局開始時的發球權歸上一局比賽的接球手，該名運動員在自己的右邊發球區發球，然後將發球權交給對手。接下來雙方在每兩次發球後便交換發球權。 每名選手在其每兩次發球中，先在左邊發球區發球，再在右邊發球區發球。每次比數總和達至6的倍數後， (e.g. when the score is 4-2)，雙方須交換場地。決勝局完結後，雙方亦須交換場地，因為決勝局必定是單數局 7-6（一盤中的第13局）。

    場 match

    大多數比賽為 3 盤 2 勝制，採決賽局／盤制（tie-break set），先贏得 2 盤即贏得比賽；大滿貫男子單打比賽為 5 盤 3 勝制。

    大滿貫賽制

    從2019開始，目前四大滿貫決勝盤（final set）賽制如下：

  • 澳網：先贏 6 局者獲勝，如果雙方打到 5-5 的情況下，先贏 7 局者獲勝；如果雙方打到6-6 平局時進行搶十大戰。
  • 法網：領先盤（advantage set）決勝。
  • 溫網：先贏 12 局者獲勝，如果雙方打到 11-11 的情況下，先贏13局者獲勝；如果雙方打到 12-12 平局時進行搶七大戰。
  • 美網：與常規盤一樣，搶七決勝。
